Colorectal Cancer Variant Atlas (CRC-VA)

The Colorectal Cancer Genomic Variant Database (CC-GVD) is a comprehensive repository of germline and somatic variants identified from colorectal cancer (CRC) exome datasets, along with normal colorectal tissue exome datasets. The database is built from high-throughput sequencing data processed using an automated Next-Generation Sequencing (NGS) pipeline, ensuring accurate variant calling and annotation.

Dataset Composition:
  • Exome data from colorectal cancer patients and normal tissue samples.
  • Germline variants identified using the GATK Best Practices pipeline.
  • Somatic mutations detected using Mutect2, a state-of-the-art tool for distinguishing tumor-specific mutations.
Pipeline Overview:
  • Raw sequencing reads are retrieved from NCBI-SRA and undergo preprocessing.
  • Quality control is performed using FastQC & MultiQC.
  • Alignment is conducted using Bowtie2, followed by conversion to BAM format using SAMtools.
  • Variants are called with GATK (HaplotypeCaller for germline) and Mutect2 (for somatic variants).
  • Annotation is performed using SnpEff.
  • Functional impact is assessed using SIFT & PolyPhen2.
  • Variants are prioritized based on frequency and potential pathogenicity.
  • Gene expression analysis integrates data from GEPIA2, OncoMX, and cBioPortal.
Applications:
  • Identification of CRC-specific somatic mutations.
  • Discovery of novel germline risk variants.
  • Comparative analysis between tumor and normal tissue mutations.
  • Support for biomarker discovery and targeted therapy research.
